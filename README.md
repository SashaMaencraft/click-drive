## 🎮 О ЧЁМ ИГРА?

Ты — **КРИК-ИНЖЕНЕР**. Твоя задача — разбудить вселенную своим криком.  
Но кричать просто так нельзя — нужны **батареи**, которые сами восстанавливаются.  
Копишь энергию (CRY), покупаешь планеты, улучшения и орёшь громче всех в галактике.

Игровой процесс построен на **механике кликов с ограниченным зарядом** — батарея тратится на каждый крик, но восстанавливается со временем.

---

## 🧠 ЛОГИКА ИГРЫ (как это работает)

### 🔋 Батарея

- У тебя есть заряд батареи (макс. 100 → потом увеличивается).
- Один клик = минус 1 заряд.
- Если заряд = 0 → кликать **нельзя** (кнопка блокируется).
- Батарея **восстанавливается сама**:
  - по умолчанию: 1 заряд каждые 2 секунды
  - можно купить улучшение регена (1/сек, 2/сек, 4/сек)

### 💰 Валюта — CRY (Койны)

- 1 клик = 1 CRY (с бонусами может быть больше).
- CRY тратятся в магазине на батареи, улучшения и покупку планет.

### 🚀 Прогресс по планетам

| Уровень | Планета | Как открыть |
|---------|---------|--------------|
| 0 | 💫 Рождение | стартовая планета |
| 1 | 🌍 Земля | купить за **500 CRY** |
| 2 | 🌙 Луна | купить **Тюнинг Плакса (2900) + Смазку (2100)** |
| 3 | 🔴 Марс | купить **Вой-Вольт (4900) + Кристалл воя (4700)** |
| 4 | ☀️ Солнце | купить **Мега-батарею (15000)** |
| 5 | ⭐ Звезда | финал, доступен после покупки Мега-батареи |

> ⚠️ **Важно:** каждая планета открывается **только после выполнения условий**. Ничего не даётся просто так.

### 🛒 Что можно купить в магазине

- 🔋 **Малый Рык** (+35 заряда) — 90 CRY  
- 🔊 **Басс-ярость** (+320 заряда) — 750 CRY  
- 🎺 **Тюнинг «Плакса»** (шанс +50 CRY при клике) — 2900 CRY  
- ⚙️ **Смазка пальцев** (шанс +2 CRY) — 2100 CRY  
- ♾️ **Вой-Вольт** (800 заряда + перезарядка 2 мин) — 4900 CRY  
- 💎 **Кристалл воя** (+1 CRY/сек пассивно) — 4700 CRY  
- ⭐ **Мега-батарея** (+1000 заряда) — 15000 CRY  
- 🤖 **Автокликер** (1 клик/сек без траты батареи) — 5000 CRY  
- 🎯 **Улучшение крита** (шанс x2 награды) — 3000 CRY  
- 🔋 **Улучшение регена** (3 уровня: 1/сек, 2/сек, 4/сек) — 800 / 2500 / 6000 CRY  

### ⚡ Ускорение (Boost)

- **1-я покупка:** 200 CRY  
- **2-я покупка:** 500 CRY  
- **3+ покупка:** 1200 CRY  
- **Эффект:** 60 секунд каждый клик даёт **x3 CRY**  
- **Важно:** ускорение работает **даже при разряженной батарее**!

### 📊 Система уровней

- Уровень пилота растёт от:
  - количества открытых планет
  - каждые 100 кликов = +1 уровень  
- Макс. уровень: 20  

### 👤 Профиль

- Нажимай на аватарку в правом верхнем углу → открывается модальное окно со статистикой:
  - уровень
  - всего кликов
  - всего заработано CRY
  - прогресс по планетам
  - версия игры

### 💾 Сохранение и сброс

- **Автосохранение** каждые 5 секунд (в `localStorage`).
- **Кнопка «ПОЛНЫЙ СБРОС»** — стирает всё: планеты, CRY, улучшения, клики, батарею. После сброса ты снова в «Рождении» с 0 CRY.

## 🎯 ЦЕЛЬ ИГРЫ

1. Купить **Землю** (500 CRY)
2. Открыть **Луну** (купить Плаксу + Смазку)
3. Открыть **Марс** (купить Вой-Вольт + Кристалл)
